# Ngram Datasets

This folder contains datasets derived from **Google Ngram Viewer**, used to analyze the historical frequency of selected artistic terms in printed sources.

Two distinct data collections were produced for this project, each addressing a different analytical question in the dissertation.

---

## Dataset structure

The folder is divided into two subdatasets:

### `cornell_rauschenberg`

This dataset tracks the frequency of the names **Robert Rauschenberg** and **Joseph Cornell** between **1945 and 1960**.

The goal of this collection is to examine the emergence and circulation of these artists’ names in printed discourse during the early postwar period.

Detailed documentation is available in:

`ngrams/cornell_rauschenberg/README.md`

---

### `ismos`

This dataset tracks the frequency of three art movements:

* **Cubism**
* **Surrealism**
* **Impressionism**

between **1875 and 1925**.

The purpose of this dataset is to analyze the historical diffusion of these artistic categories in printed sources.

Detailed documentation is available in:

`ngrams/ismos/README.md`

---

## Notes

Each subdataset contains its own:

* dataset files (`data`)

* visualizations (`visualizations`)
* specific methodological documentation (`README.md`)

Users interested in methodological details should consult the README file within each subfolder.
