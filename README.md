# A biografia do ready-made, da natureza-morta a Rauschenberg

**Autor:** Felipe Eduardo Lázaro Braga  
**Instituição:** Universidade de São Paulo (USP)  
**Programa:** Sociologia  
**Ano:** 2026  

## 📘 Descrição

Este repositório contém os bancos de dados, os scripts em R e as visualizações utilizados na tese *A biografia do ready-made, da natureza-morta a Rauschenberg*.  

O objetivo é garantir a reprodutibilidade das análises, disponibilizar publicamente os dados utilizados na pesquisa e documentar os procedimentos metodológicos adotados.

A tese articula três eixos analíticos principais:

- análise de obras de arte  
- análise de redes de artistas  
- análise textual por meio de n-grams  

Cada eixo possui um módulo próprio neste repositório, contendo dados, scripts e resultados.

## 🧱 Estrutura do repositório

- `obras-arte/` → banco de dados, scripts e visualizações das obras analisadas  
- `redes-artistas/` → construção e análise das redes de artistas  
- `ngrams/` → processamento textual e extração de n-grams  
- `docs/` → descrição metodológica geral  

Cada pasta contém um `README.md` com instruções específicas de reprodução.

## 🔁 Reprodutibilidade

Versão do R utilizada: **4.5.2 (2025-10-31 ucrt)**  

Para reproduzir as análises:

1. Instalar o R na versão indicada (ou compatível)  
2. Instalar os pacotes necessários (listados nos scripts ou via `renv`, se disponível)  
3. Executar os scripts na ordem indicada em cada módulo  
4. Os outputs (gráficos e tabelas) serão gerados automaticamente na pasta `visualizations/` correspondente  

## 📊 Visualizações

As principais visualizações produzidas na tese serão disponibilizadas na pasta `visualizations/` de cada módulo e exemplos serão exibidos aqui após o upload dos gráficos.
